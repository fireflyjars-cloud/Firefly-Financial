# Firefly Financial — Vision & Roadmap

This document is for internal planning only.

## Phase 1 — Web App (Current)
- Manual entry, no bank connections
- ASMR pour animation
- 12-level journey
- Streaks, Light Coins, weekly quests
- Community feed and content hub
- AI assistant trained on the Jarring Framework

## Phase 2 — 6 to 18 Months
- Read-only bank connection via Plaid
- AI agents for lead nurturing
- Expanded community features

## Phase 3 — 18+ Months
- Real money movement
- Firefly Houses
- Firefly University
- Native iOS and Android
